# 💀 Death By Sweets

A gothic-cute bakery website for Death By Sweets — handcrafted cookies, brownies, cupcakes, cakes, and loaves.

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Create a new GitHub repo called `Death-By-Sweets`
2. Push this code to the `main` branch
3. Go to **Settings → Pages → Source** and select **GitHub Actions**
4. The workflow will auto-deploy on every push to `main`

> **Important:** If your repo name is different from `Death-By-Sweets`, update the `base` value in `vite.config.ts` to match: `base: '/your-repo-name/'`

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
